Control Butttons:
Run Codes: see code output in resutl screen, run code support javascript,css,html codes.
Dark mode: changed theme color into dark mode, this effect changed all page into dark mode without result screen.
Light mode: changed theme color into light mode, this effect changed all page into light mode without result screen,it also set by defulte.
Clean Output: clean output screen result, and keep codes on screen.
Save Codes: save input codes into cache memory,it run by defulte after refresh the page.
Delete Cache: delete all cached memory,it show empty screen by defulte.
Refresh Page: refresh the page and put saved codes from the cache memory.
